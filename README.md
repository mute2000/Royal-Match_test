  该项目包含我对Royal-Match游戏玩法机制的理解和核心玩法手动测试的实践。

## 游戏理解
  1. 其中包括我对游戏核心机制和活动机制的理解和玩过这款游戏后我自己的感受。
  2. 具体内容看： https://github.com/mute2000/Royal-Match_test/blob/master/%E6%B8%B8%E6%88%8F%E7%90%86%E8%A7%A3.md
